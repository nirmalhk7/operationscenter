# Tool Boundary

`eq_thesis_depth_reviewer` is a read-only MountainValue subagent role.

Allowed behavior:
- Review the narrowed candidate JSON and prior reviews.
- Use primary-source facts supplied in the payload.
- Return compact JSON.

Disallowed behavior:
- Discord or forum posting.
- Scheduling or cron changes.
- Trade execution.
- Inventing missing price, enterprise value, filing, transcript, or management
  evidence.
