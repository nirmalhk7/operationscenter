# Operations Center

GitOps-managed Paperclip company for coordinating OpenClaw agents in this repository.

## Goal

Run infrastructure, research, and operations work through OpenClaw agents with ticket-based orchestration, approvals, and audit trails.

## Source of truth

- Agent runtime config: `openclaw/openclaw.json`
- Prompt artifacts: `openclaw/mainAgent/`, `openclaw/customAgents/`, `openclaw/subAgents/`
- Company package: `clusters/managed/llm/paperclip/company/operations-center/`
